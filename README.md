# Mading Digital Kelas📰✨
**Mading Digital** adalah aplikasi web berbasis frontend yang berfungsi sebagai papan majalah dinding (mading) digital. Website ini dirancang untuk menampilkan artikel, pengumuman, dan galeri kegiatan sekolah dalam tampilan modern, bersih, dan responsif.

🌐 **Live Demo:**  
https://kabureneo.github.io/MadingDigital

---

## 🚀 Tentang Proyek Ini
Mading Digital adalah **project pertama saya dalam pengembangan web frontend**.  
Proyek ini dibuat dengan semangat belajar, eksplorasi, dan *vibe coding* — fokus membangun, mencoba, memperbaiki, dan terus mengembangkan ide sampai menjadi tampilan web yang fungsional dan menarik.

Project ini menjadi fondasi awal perjalanan saya di dunia web development.

---

## 🎯 Tujuan Proyek
- Menghadirkan versi digital dari mading sekolah
- Melatih kemampuan HTML, CSS, dan JavaScript
- Membangun UI yang interaktif dan responsif
- Menjadi dasar pengembangan sistem mading berbasis database di masa depan
- Membangun kebiasaan menulis kode yang rapi dan terstruktur

---

## ✨ Fitur Utama
- Halaman utama dengan artikel unggulan
- Kategori konten (Pendidikan, Seni, Olahraga, Teknologi, dll.)
- Galeri foto kegiatan sekolah
- Bagian pengumuman
- Navigasi responsif
- Animasi scroll dan efek interaktif
- Desain modern dan mobile-friendly

---

## 🛠 Teknologi yang Digunakan
- **HTML5** — Struktur halaman  
- **CSS3** — Styling dan layout  
- **JavaScript (Vanilla JS)** — Interaktivitas dan animasi  

Seluruh sistem berjalan di sisi client (frontend) tanpa framework dan tanpa backend.

---

## 📁 Struktur Folder
Berikut struktur dasar proyek:
MadingDigital/
│
├── index.html # Halaman utama website
├── style.css # File styling dan desain tampilan
├── script.js # File JavaScript untuk interaksi & animasi

Struktur dibuat sederhana agar mudah dipahami dan dikembangkan lebih lanjut.

## 🔮 Rencana Pengembangan
Ke depan, proyek ini dapat dikembangkan menjadi sistem yang lebih kompleks, seperti:

- Integrasi backend (Node.js / PHP / Firebase)
